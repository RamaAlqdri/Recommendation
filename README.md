# Recommendation Scoring Project

Repositori ini berisi project end-to-end machine learning untuk prediksi skor rekomendasi produk e-commerce.

Lokasi source code utama:
- `project/`

Dokumentasi lengkap dari awal sampai menjalankan API:
- [project/README.md](project/README.md)

Quick start:

```bash
cd project
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python train.py
python app.py
```
